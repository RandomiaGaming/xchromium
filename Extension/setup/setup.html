Ungoogled Chromium Setup Instructions:


Step 1 - Chrome Webstore:

First change the flag extension-mime-request-handling to "Always prompt for install".

Then click this link and press "Add extension" when asked if you would like to install the chromium web store.


Step 2 - DuckDuckGo:

Click here to open the search engine settings.

Under site search press add and then type the following:
Name: DuckDuckGo
Shortcut: duckduckgo.com
URL with %s in place of query: https://start.duckduckgo.com/?q=%s&kae=d&kp=1&kbe=0&kbg=-1&k5=2&kz=-1&k1=-1&kak=-1&kax=-1&kaq=-1&kao=-1&kau=-1&kap=-1&kpsb=-1&kav=1&k18=1
Suggestions URL with %s in place of query: https://start.duckduckgo.com/ac/?q=%s&type=list&kae=d&kp=1&kbe=0&kbg=-1&k5=2&kz=-1&k1=-1&kak=-1&kax=-1&kaq=-1&kao=-1&kau=-1&kap=-1&kpsb=-1&kav=1&k18=1
